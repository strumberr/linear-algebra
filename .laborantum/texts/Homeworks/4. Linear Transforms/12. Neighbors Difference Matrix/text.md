# Neighbors Difference Matrix

Write a code that generates a matrix that computes a vector 
with difference of adjacent elements of the original vector 
order of coordinates of a vector in $N$-dimenional space:

$$
\begin{bmatrix}
x_1 \\
\vdots \\
x_N
\end{bmatrix}
\rightarrow
A
\rightarrow
\begin{bmatrix}
x_2 - x_1 \\
\vdots \\
x_N - x_{N-1}
\end{bmatrix}
$$
